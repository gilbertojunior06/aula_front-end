Glossário de Tecnologias Web

HTML
Linguagem de marcação usada para estruturar o conteúdo de páginas web.
CSS
Folhas de estilo em cascata que definem a aparência e o design de páginas web.
JavaScript (JS)
Linguagem de programação que adiciona interatividade e dinamismo às páginas web.
HTTP
Protocolo de comunicação usado para transferir dados entre cliente e servidor na web.
API
Conjunto de definições e protocolos que permite a integração entre diferentes sistemas e aplicações.
